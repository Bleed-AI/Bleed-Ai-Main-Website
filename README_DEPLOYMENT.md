## ⚡ Quick Deployment (5 Minutes)

### Option 1: File Manager (Easiest)
```
1. Log into Hostinger → File Manager
2. Navigate to public_html
3. Upload bleedai-hostinger-deploy.tar.gz
4. Extract the file
5. Move contents to public_html root
Done!
```

### Option 2: FTP (Fastest)
```
1. Get FTP credentials from Hostinger
2. Use FileZilla (free download)
3. Upload all files from out/ to public_html/
Done!
```

### Option 3: Direct Copy
```
1. Copy the entire "out" folder
2. Upload directly to public_html
3. Make sure files are in root, not in subfolder
Done!
```
